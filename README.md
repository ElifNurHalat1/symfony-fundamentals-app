# Symfony Fundamentals App

This project was developed while following a Symfony course.
It demonstrates core Symfony concepts and best practices.

## Topics Covered
- Installation & built-in web server
- MVC architecture
- Controllers & Routing
- Twig templates
- Doctrine ORM
- Database migrations
- Request & Response handling
- Validation
- Flash messages
- Symfony Profiler

## Tech Stack
- PHP 8.x
- Symfony
- Twig
- Doctrine ORM
- MySQL

## Purpose
The purpose of this project is to demonstrate hands-on experience with Symfony fundamentals
and to serve as a reference project for backend development using Symfony.
